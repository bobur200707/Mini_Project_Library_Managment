# Library Management System (Frontend Only)

## Overview
This is a **frontend-only Library Management System** built using **HTML, CSS, Bootstrap, and JavaScript**.  
The system allows you to manage books, members, and issue/return transactions completely in the browser without a backend.

---

## Features

### Books
- Add new books (name, author, genre, quantity)  
- Delete books  
- Search books by name  
- Default books included:
  1. The Great Gatsby
  2. To Kill a Mockingbird
  3. 1984
  4. The Catcher in the Rye

### Members
- Add new members (name, email, phone)  
- Delete members  
- Default members included:
  1. Alice Johnson
  2. Bob Smith
  3. Charlie Brown

### Issue / Return
- Issue book to a member  
- Return book  
- List of all issued books with date  
- Dynamic dropdowns for available books and members  

---

## How to Use
1. Open `index.html` in any modern browser (Chrome, Firefox, Edge, etc.)  
2. Navigate between **Books**, **Members**, and **Issue/Return** sections using the navbar  
3. Add new books or members using the forms provided  
4. Issue a book by selecting a book and a member, then click "Issue Book"  
5. Return a book using the "Return" button in the issued books list  
6. Search for books using the search input on the **Books** page  

---

## Technologies Used
- **HTML5** – structure  
- **CSS3** – styling  
- **Bootstrap 5** – responsive design  
- **JavaScript (ES6)** – interactivity  

---

## Notes
- This project is **frontend only**; all data is stored in memory and will reset on page refresh  
- Can be extended with backend and database for persistence  
- Default books and members are preloaded for demo purposes
